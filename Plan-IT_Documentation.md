# Plan-IT

### Project Overview
Plan-IT is an internal workplace productivity ecosystem designed for modern companies. It replaces fragmented tools like Microsoft Planner with one smart, engaging, zero-cost platform that unifies project management, team collaboration, performance tracking, and resource documentation.

### Problem Statement
Modern companies struggle to maintain employee engagement and clear accountability with current corporate tools. The standard tools are often viewed as chore-like, disconnected, and lack a clear incentive structure for employees to actively update their status or share knowledge. 

### Pain Points
1. Employees do not show interest in updating their planners currently.
2. The current system is not transparent, leading to silos.
3. Project resources are scattered and not stored in one united place.
4. An anonymous feedback system does not exist, stifling honest communication.
5. HR manually messages employees monthly for updates.
6. The current Microsoft Planner costs money and is not cost-effective.
7. Existing systems are boring and result in low engagement.
8. Employees lack motivation.
9. No clear reward system.
10. No employee recognition model.

### Features
- **Smart Task Management**: Gamified experience with difficulty and urgency based XP.
- **Unified Projects & Resources**: Keep code, Figma files, and docs directly inside project milestones.
- **Anonymous Feedback**: Let employees voice concerns and ideas securely and invisibly.
- **Community Feed**: Celebrate wins, ask questions, and build office culture.
- **Automated HR Reporting**: Export tasks, performance, and project sheets to Excel/CSV in one click.
- **Performance Engine**: Intelligent scoring system calculating efficiency, quality, and collaboration.

### Module Breakdown
- **Dashboard**: High-level overview, deadlines, rank, Performance Score, and Employee of the Month.
- **My Tasks**: Unified task view with comprehensive filtering and queue management.
- **Projects**: Centralized timelines and milestones.
- **Resources**: Global and project-specific file storage.
- **Team**: Employee directory.
- **Community**: Feed-style updates and social engagement.
- **Leaderboard**: Weekly, monthly, and overall rankings.
- **Reports**: Data export for HR.
- **Feedback**: Anonymous submission portal.

### Workflow
1. **Login**: Users authenticate securely via Microsoft OAuth.
2. **Dashboard Review**: See what is pending, what is overdue, and upcoming milestones.
3. **Execution**: Pick a task, mark it for verification upon completion.
4. **Approval**: Project Manager/Mentor reviews and approves, awarding XP.
5. **Collaboration**: Upload design/code resources, post wins to the Community.
6. **Reporting**: HR generates one-click monthly Excel reports on task adherence and scores.

### User Roles
- **Employee**: Completes tasks, submits feedback, engages in community.
- **Manager/Mentor**: Approves tasks, oversees projects, manages resources.
- **HR/Admin**: Exports reports, moderates community, reviews anonymous feedback.

### Benefits
- Fun and engaging gamified experience.
- Easy to use interface.
- Free internal system compared to expensive per-seat SaaS tools.
- Transparent leaderboard fosters healthy competition.
- Better visibility and accountability across all departments.

### ROI
- Drastic reduction in software licensing costs (Microsoft Planner, Trello, Jira).
- Saved HR hours via automated reporting.
- Improved retention through better office culture and recognition.

### Security
Plan-IT uses one-way encryption for the feedback module and secure Microsoft OAuth protocols, ensuring sensitive data is protected while maintaining accessibility.

### Future Scope
